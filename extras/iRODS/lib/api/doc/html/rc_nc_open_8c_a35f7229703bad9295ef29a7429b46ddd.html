rcNcOpen
int rcNcOpen ( rcComm_t *  conn,
ncOpenInp_t *  ncOpenInp,

Description:
open an iRODS data object for netcdf operation (equivalent to nc_open).
Primary User:
client
Category:
NETCDF operations
Since:
3.1
Author:
Mike Wan
Date:
2012
Remarks:
none
Note:
none
Example Usage:

nc_open a data object /myZone/home/john/myfile.nc for write:
ncOpenInp_t ncOpenInp;
int ncid = 0;
int status;
bzero (&ncOpenInp, sizeof (ncOpenInp));
rstrcpy (ncOpenInp.objPath, "/myZone/home/john/myfile.nc", MAX_NAME_LEN);
ncOpenInp.mode = NC_NOWRITE;
status = rcNcOpen (conn, &ncOpenInp, &ncid);
if (status < 0) {
.... handle the error
}

Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]ncOpenInp- Elements of ncOpenInp_t used :
  • char objPath[MAX_NAME_LEN] - full path of the data object.
  • int mode - the mode of the open - valid values are given in netcdf.h - NC_NOWRITE, NC_WRITE
[out]ncid- the ncid of the opened object.

integer
