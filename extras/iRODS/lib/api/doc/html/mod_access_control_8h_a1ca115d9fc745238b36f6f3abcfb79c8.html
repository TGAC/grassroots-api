Example Usage:

Modify object access control:
See ichmod.c as it sets fields and calls rcModAccessControl to perform
the various functions of ichmod (see 'ichmod -h).
For example:

modAccessControlInp.recursiveFlag = myRodsArgs.recursive;
modAccessControlInp.accessLevel = argv[myRodsArgs.optind];
modAccessControlInp.userName = userName;
modAccessControlInp.zone = zoneName;
modAccessControlInp.path = rodsPathInp.srcPath[0].outPath;
status = rcModAccessControl(conn, &modAccessControlInp);
if (status < 0) {
.... handle the error
}

[in]conn- A rcComm_t connection handle to the server.
