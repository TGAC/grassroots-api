rcNcInqId
int rcNcInqId ( rcComm_t *  conn,
ncInqIdInp_t *  ncInqIdInp,
int **  outId 
)
Description:
General netcdf inquiry for id (equivalent to nc_inq_dimid, nc_inq_varid, .... This API is superceded by the more comprehensive rcNcInq API.
Primary User:
client
Category:
NETCDF operations
Since:
3.1
Author:
Mike Wan
Date:
2012
Remarks:
none
Note:
none
Example Usage:

nc_open a data object and get the dimension Id of 'time':
ncOpenInp_t ncOpenInp;
ncInqIdInp_t ncInqIdInp;
int *outId;
int ncid = 0;
ncInqWithIdOut_t *ncInqWithIdOut = NULL;
int status;
bzero (&ncOpenInp, sizeof (ncOpenInp));
rstrcpy (ncOpenInp.objPath, "/myZone/home/john/myfile.nc", MAX_NAME_LEN);
ncOpenInp.mode = NC_NOWRITE;
status = rcNcOpen (conn, &ncOpenInp, &ncid);
if (status < 0) {
.... handle the error
}
bzero (&ncInqIdInp, sizeof (ncInqIdInp));
ncInqIdInp.ncid = ncid;
ncInqIdInp.paramType = NC_DIM_T;
rstrcpy (ncInqIdInp.name, "time", MAX_NAME_LEN);
status = rcNcInqId (conn, &ncInqIdInp, &outId);
if (status < 0) {
.... handle the error
}
ncInqIdInp.myid = *outId;
status = rcNcInqWithId (conn, &ncInqIdInp, &ncInqWithIdOut);
if (status < 0) {
.... handle the error
}

Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]ncInqIdInp- Elements of ncInqIdInp_t used :
  • int paramType - what to inquire - valid values are defined in ncInqId.h - NC_VAR_T or NC_DIM_T.
  • int ncid - the ncid from ncNcOpen.
  • char name[MAX_NAME_LEN] - the name of the item to inquire.
[out]outId- the returned id of the inquiry

integer
