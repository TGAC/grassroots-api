rcNcRegGlobalAttr

Description:
Extract the NETCDF global variables in an iRODS data object and register them as AUV.
Primary User:
client
Category:
NETCDF operations
Since:
3.1
Author:
Mike Wan
Date:
2012
Remarks:
none
Note:
none
Example Usage:

Extract and register all global attributes of "/myZone/home/john/myfile.nc"
ncRegGlobalAttrInp_t ncRegGlobalAttrInp;
int status;
bzero (&ncRegGlobalAttrInp, sizeof (ncRegGlobalAttrInp));
rstrcpy (ncRegGlobalAttrInp.objPath, "/myZone/home/john/myfile.nc", MAX_NAME_LEN);
status = rcNcRegGlobalAttr (conn, &ncRegGlobalAttrInp, &ncid);
if (status < 0) {
.... handle the error
}

Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]ncRegGlobalAttrInp- Elements of ncRegGlobalAttrInp_t used :
  • char objPath[MAX_NAME_LEN] - full path of the data object.
  • numAttrName - number of attributes to extract and register. If it is 0, ALL attributes will be done.
  • char **attrNameArray - An array of numAttrName pointers containing the atrribute names.
[out]none
