rcNcInq
int rcNcInq ( rcComm_t *  conn,
ncInqInp_t *  ncInqInp,
ncInqOut_t **  ncInqOut 
)
Description:
General netcdf inquiry (equivalent to nc_inq + nc_inq_format). Get all info including attributes, dimensions and variables assocaiated with a NETCDF file will a single call.
Primary User:
client
Category:
NETCDF operations
Since:
3.1
Author:
Mike Wan
Date:
2012
Remarks:
none
Note:
none
Example Usage:

nc_inq an opened data object and print the header info:
ncOpenInp_t ncOpenInp;
ncInqInp_t ncInqInp;
int ncid = 0;
int status;
bzero (&ncOpenInp, sizeof (ncOpenInp));
rstrcpy (ncOpenInp.objPath, "/myZone/home/john/myfile.nc", MAX_NAME_LEN);
ncOpenInp.mode = NC_NOWRITE;
status = rcNcOpen (conn, &ncOpenInp, &ncid);
if (status < 0) {
.... handle the error
}
bzero (&ncInqInp, sizeof (ncInqInp));
ncInqInp.ncid = ncid;
ncInqInp.paramType = NC_ALL_TYPE;
ncInqInp.flags = NC_ALL_FLAG;
status = rcNcInq (conn, &ncInqInp, &ncInqOut);
.... handle the error
if (status < 0) {
}
status = prNcHeader (conn, ncid, False, ncInqOut);

Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]ncInqInp- Elements of ncInqInp_t used :
  • int paramType - what to inquire - valid values are defined in ncInq.h - NC_VAR_TYPE, NC_DIM_TYPE, NC_ATT_TYPE and NC_ALL_TYPE.
  • int ncid - the ncid from ncNcOpen.
  • int myid - the id (from rcNcInqId) to inquire. Meaning only if NC_ALL_TYPE == 0.
  • char name[MAX_NAME_LEN] - the name of the item to inquire. Either myid or name, but not both, can be input. Meaning only if NC_ALL_TYPE == 0.
[out]ncInqOut- a ncInqOut_t. Elements of ncInqOut_t:
  • int ndims - number of dimensions
  • ncGenDimOut_t *dim - an array with ndims length of ncGenDimOut_t struct
  • int ngatts - number of global attributes
  • ncGenAttOut_t *gatt - an array with ngatts length of ncGenAttOut_t struct
  • int nvars - number of variables
  • ncGenAttOut_t *var - an array with nvars length of ncGenVarOut_t struct
  • int unlimdimid - the dimid of the unlimited dimension
  • int format - NC_FORMAT_CLASSIC, NC_FORMAT_64BIT, NC_FORMAT_NETCDF4 or NC_FORMAT_NETCDF4_CLASSIC

integer
