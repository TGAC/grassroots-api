Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]procStatInp- Elements of procStatInp_t used :
  • char addr[LONG_NAME_LEN] - get the clients running only on this server. If empty, get the clients on the connected server.
  • char rodsZone[NAME_LEN] - get the clients running in this zone.
  • keyValPair_t condInput - keyword/value pair input. Valid keywords:
    RESC_NAME_KW - get the clients on this resource server.
    ALL_KW - get the clients on all servers. This keyword has no value.
[out]procStatOut- arrays of client results given in a genQueryOut_t. The index identifying the result arrays are:
PID_INX - the pid of the agent
STARTTIME_INX - the start time
CLIENT_NAME_INX - the client user name
CLIENT_ZONE_INX - the client user zone
PROXY_NAME_INX - the proxy user name
PROXY_ZONE_INX - the proxy user zone
REMOTE_ADDR_INX - the client address
PROG_NAME_INX - the client process name
SERVER_ADDR_INX - the server address
