rcModAVUMetadata
rcModAVUMetadata ( rcComm_t *  conn,
modAVUMetadataInp_t *  modAVUMetadataInp 
)
Description:
Modify Attribute-Value-Unit information associated with an object;
the user-defined metadata.
Primary User:
clients, in the 'C' code this is used by 'imeta'
Category:
metadata operations
Since:
1.0
Author:
Wayne Schroeder
Date:
2007
Remarks:
none
Note:
none
Example Usage:

Modify AVUs:
See imeta.c as it sets fields and calls rcModAVUMetadata to perform
the various functions of imeta (see 'imeta -h'). rsModAVUMetadata
calls chlAddAVUMetadata, chlDeleteAVUMetadata, chlCopyAVUMetadata or
chlModAVUMetadata.
Example client call, corresponding to
'imeta add -d f1 attrName attrValue':
modAVUMetadataInp.arg0 = "add";
modAVUMetadataInp.arg1 = "-d"
modAVUMetadataInp.arg2 = "/tempZone/home/rods/f1";
modAVUMetadataInp.arg3 = "attrName"
modAVUMetadataInp.arg4 = "attrValue";
modAVUMetadataInp.arg5 = "";
modAVUMetadataInp.arg6 = "";
modAVUMetadataInp.arg7 = "";
modAVUMetadataInp.arg8 = "";
modAVUMetadataInp.arg9 = "";
status = rcModAVUMetadata(conn, &modAVUMetadataInp);
if (status < 0) {
.... handle the error
}

[in]conn- A rcComm_t connection handle to the server.
