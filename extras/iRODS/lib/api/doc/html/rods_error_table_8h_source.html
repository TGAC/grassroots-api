00007 /* error code format:
00008  *
00009  *      -mmmmnnn
00010  *
00011  * where -mmmm000 is an error code defined in the rodsErrorTable.h to define
00012  * an error event. e.g.,
00013  *
00014  * #define SYS_SOCK_OPEN_ERR               -1000
00015  *
00016  * which define an error when a socket open call failed. Here mmmm = 1
00017  *
00018  * nnn is the errno associated with the socket open call. So, if the errno
00019  * is 34, then the error returned to the user is -1034. We use 3 figures
00020  * for nnn because the errno is less than 1000.
00021  */
00022
