collEnt_t
collEnt_t
Description:
Output structure for collection read operation. Each structure contains info on an object read.
Since:
1.0
Remarks:
none
Note:
Elements of collEnt_t:
  • objType_t objType - the type of object. Valid types are:
    DATA_OBJ_T
    COLL_OBJ_T
  • int replNum - the replica number if a DATA_OBJ_T
  • int replStatus - the replica status if a DATA_OBJ_T
  • uint dataMode - the mode if a DATA_OBJ_T
  • rodsLong_t dataSize - the data size if a DATA_OBJ_T
  • char *collName - full path of the collection if a COLL_OBJ_T
    full path of the parent collection if a DATA_OBJ_T
  • char *dataName - the name of the data object if a DATA_OBJ_T
  • char *dataId - the data ID if DATA_OBJ_T
  • char *createTime - creation time in seconds since Epoch
  • char *modifyTime - the last modified time
  • char *chksum - the md5 checksum value if a DATA_OBJ_T
  • char *resource - the resource where the data object is located
  • char *rescGrp - the resource group
  • char *phyPath - the full physical path of the file in resource vault
  • char *ownerName - the owner of the data object or collection
  • specColl_t specColl - valid only if the path is in a special collection (e.g., mounted collection).
