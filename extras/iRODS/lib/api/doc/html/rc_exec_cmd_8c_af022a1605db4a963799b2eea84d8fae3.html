rcExecCmd

execCmd_t *  execCmdInp,

Description:
Execute a command stored in the /server/bin/cmd directory of a iRODS server.
Primary User:
client
Category:
misc operations
Since:
1.0
Author:
Mike Wan
Date:
2007
Remarks:
none
Note:
none
Example Usage:

Execute the "hello" command on the server. The "hello" command is a simple script store in the /server/bin/cmd directory of a server.
int status;
execCmd_t execCmdInp;
execCmdOut_t *execCmdOut = NULL;
bzero (&execCmdInp, sizeof (execCmdInp));
rstrcpy (execCmdInp.cmd, "hello", LONG_NAME_LEN);
rstrcpy (execCmdInp.cmdArgv, "John", LONG_NAME_LEN);
if (status < 0) {
.... handle the error
}

Parameters:
[in]conn- A rcComm_t connection handle to the server.
[in]execCmdInp- Elements of execCmd_t used :
  • char cmd[LONG_NAME_LEN] - The cmd to execute.
  • char cmdArgv[LONG_NAME_LEN] - The input argument for the cmd.
  • char execAddr[LONG_NAME_LEN] - The address of the server to execute this cmd. This input can be empty
  • char hintPath[MAX_NAME_LEN] - Execute where this file is located.
  • int addPathToArgv - whether to add the resolved path to the argv.
[out]execCmdOut- A execCmdOut struct for the output.

integer
